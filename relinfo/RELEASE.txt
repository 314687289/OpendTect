OpendTect V5.0.x : Release info

----
v5.0.9: August 11, 2015
 * Fix for crash while duplicating an already existing volume in the tree
 * Fix for crash while creating Multicube Prestack 3D data
 * Fix for crash while loading 2D line
 * Fix for crash while creating new interactive random line
 * Fix for blank progress window for batch processing if data folder has a
   space in its name
 * Disabled displaying duplicate volumes
 * Displaying only dGB license fields on MAC platform
 * Fix for reversed axis in 2D viewer
 * 'Remove Selection' button is active only if the polygon selection
   tool is active
 * Fix for freezing of OpendTect after zooming volume and
   multi-attribute inline
 * Fix for removing pick using 'Ctrl+Left-mouse-button' while creating
   new well track
 * Fix for scene annotation color update
 * Fix for latitude/longitude to coordinate calculation when survey
   XY in feet
 * Fix for saving properties of gather width and view side after clicking on
  'OK' in PreStack display properties window
 * Fix for correction of Time-Depth model import
 * Improved displaying faults using 'on horizons only'
 * Fix for closing PreStack display properties window(s) after removing
   PreStack data from scene
 * Fix for creating volume in selected region using volume builder tool
 * Fix for updating  grid lines after changed Z-scale
 * Fix for Random line with preloaded attribute showing wrong texture while
   pulling down z-dragger
 * Fix for displaying of wellhead values on 'Edit Well track' window
 * Fix for displaying flattened 2D at non-zero Z
 * Fixed layout issue in 'Advanced Parameters' window in Angle Mute computer
 * Fix for 'Save as' option of stratigraphic levels in Stratigraphic Framework
 * Fix for reverting back/reset of stratigraphic tree
 * Fix for lock button in Stratigraphic tree

v5.0.7: May 7, 2015
 * Horizon reading speed much improved
 * Fix for crash while creating a flattened scene from a 2D horizon if
   geometry of a 2D line is not available
 * In the session 'Save as' window, the current session is now pre-selected
 * Fix for some erroneous connections while displaying huge FaultStickSets
 * Removed support for non-standard fonts
 * Removed option to generate 3D seismics from 2D as the results were not
   satisfactory
 * Fix for disappearing of far Inline/Crossline/Z slice when zooming in
 * Fix for unusually bright texture on seismic sections at specific
   light/camera angles
 * Fix for wrong angle displayed in 'Survey setup and Selection' window
 * Increased the default value of ambient reflectivity for horizon displays
 * Fix for wrong message display while saving new random line
 * Fix for Picksets and Fault intersections stiil visible when displayed in
   'on sections only' mode and Inline/Crossline/Z sections are turned off
 * Fix for user-specified storage type being overridden by the storage type of
   the source cube while making a copy
 * Enabled filling of a body with user-defined value in Volume Builder
 * Fix for marker size being too small in large surveys
 * Fix for 'View Inline' (Crossline/Z) button always showing the default
   orientation. Now it considers the current orientation
 * It is again possible to select a Fault stick by clicking on
   one of its nodes
 * Fix for partial display of attributes while restoring sessions with
   attributes of different data sizes
 * Fix for creating log with 'Vint' option
 * Fix for copy/paste between OpendTect table and external spreadsheets
   like OpenOffice
 * Fix for crash while reopening previously saved UVQ model
 * Copying a seismic cube can now be done in batch mode
 * Fix for various issues with wrong time-depth models while importing wells
 * Removed 'SEGYDirect' option from Prestack output fields
 * Fix for displaying default values in 'Set Annotation Scale' window
 * Renamed 'RGT Data' to 'Wheeler Data' in the attribute selection menu
   inside Wheeler scenes
 * Fix for importing well markers if new list is shorter than existing list
 * Fix for missing host selection field in remote processing window if remote
   hosts are not configured
 * Fix for displaying fault intersection on horizon
 * Fix for disappearing scene while zooming in/out after removing a seed of a
   newly tracked horizon
 * Fix for random-line draggers not appearing in the scene after creating
   using interactive mode
 * Fix for non-functional 'Analyze' button in several 2D attributes

v5.0.6: March 5, 2015
 * Fix for crash while generating Depth/Time model in
   'Tie well to Seismic' window
 * Fix for defining constants with capital 'C' in Mathematics attribute
 * Fix for unit conversion and importing well track in 'Manage Wells' window

 * Fix for misleading warning messages while overwriting a horizon

 * Improved 'Display only at sections' for horizons
 * Fix for Wavelet import
 * After multi-machine processing the temporary data folder is now removed
 * Fix for attribute processing on 2D lines with trace number steps of
   more than 1

 * Made the 'Allow spin animation' option in Scene properties window work
 * Added 'Delete rows' option in well log editor

 * Improved synchronization of cursor position between scenes
 * Improved ASCII import of Well time-depth models
 * Improvements in well track import
 * Fix for crash while evaluating Mathematics attribute
 * Fix for crash while creating seismic cube with selected depth range using
   volume builder tool
 * Fix for crash while trying to save horizon without tracking anything
 * Fix for crash while importing Seismic data from simple file
 * Fix for displaying seed based iso-surface volume in scene
 * Fix for displaying of connection line between previous node and new node
   while picking fault plane
 * Fix for exporting of prestack data store into a SEGY
 * Fix for displaying of correct log units in 'Tie well to seismics' window
 * Fix for change of active color bar after saving active color bar with a
   new name
 * Fix for Splicing two logs using Mathematics engine in "Calculate new logs"

 * Fix for well name size not keeping up with snapshot resolution
 * Removed the 'Retrack all' button from Horizon tracking toolbar
 * Fix for displaying 2D lines with missing traces in 2D viewer
 * Fix for missing Random line names in restored sessions
 * Fix for tiled display of Horizon after changing Z Scale
 * Fix for creating Implicit body objects
