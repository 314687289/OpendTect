OpendTect Hardware and OS requirements

Supported platforms

OpendTect runs on 3 hardware platforms:
  1. Intel/AMD under Linux (32 or 64 bits) or MS Windows (2000 and XP)
  2. Sun/Sparc under Solaris 8 or up
  3. Mac/PowerPC under OS X (10.4.4) and up

Hardware requirements

Graphics:

OpendTect requires a recent well-patched OpenGL installation. OpenGL drivers should be updated at least every half year to ensure optimal performance and compliance.

Intel/AMD: Recent nVidia and AMD (ATI) graphics cards/chipsets. Well behaved are nVidia GeForce 6, 7 and 8 series and AMD (ATI) Radeon X1xx and X2xx series.
Mac/PowerPC:Similar to te Intel/AMD platform.
Sun/Sparc: Graphics cards like the XVR 2500 or better.

Main stream and high-end GPUs within the series are recommended, since low-level GPUs keep showing poor performances through the generations.
Since OpendTect 3.0, shading functionaly requires special GPU features, present in the cards listed above. Nevertheless, under Linux environment, only nVidia provides drivers capabilities to use the shading feature (latest so far are 100.14 drivers).

Memory

OpendTect should at least have 1 GB internal (RAM) memory available. Depending on the size of the surveys we recommend 4-8 GB on average. In special cases (big surveys or many data cubes) more memory may be required.
A rule of thumb is to have at least 10 times the displayed number of samples available. Thus, to be able to display 10 inlines with 2000 crosslines and 1000 samples per trace, you'll need a minimum of 200 million bytes of memory (i.e. 200 MB).

Processor

For Linux and MS Windows, a modern Intel or AMD processor is required. Although OpendTect will run on 2 GHz processors or even less, we recommend 3 GHz dual core for a good working environment.
The Solaris/Sparc platform performs a lot less; therefore, only high end processors like the Sun ultraSPARC IIIi and up will give tolerable performance.

Operating systems

Linux

A modern Linux distribution is required. Minimum:
  • SuSE 10.0
  • Red Hat Enterprise 3
  • Fedora core 3
For both SuSE and Red Hat-based distributions 32 or 64 bits releases are available. OpendTect is known to work under Debian, Ubuntu and other distributions, as well as earlier versions of the main distributions, too.

Solaris

Solaris 8 and up are supported.

Mac OS X

Minimum is Mac OS X (10.4.4). Mac/Intel is not supported at this point in time.

MS Windows

Windows 2000 and XP are supported. During 2007 Windows Vista compliance will be achieved.
