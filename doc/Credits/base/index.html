Introduction
OpendTect development started back in 1999 when StatoilHydro (then Statoil) requested dGB to develop a software system around the seismic object detection method the companies had jointly invented for chimney interpretation. d-Tect was developed with financial support from StatoilHydro and SenterNovem (The Dutch government).
Initially it was marketed in a conventional way as closed source system for seismic attribute processing and object detection. Late 2003 dGB adopted an Open Source business model. dTect was re-engineered and became OpendTect, an Open Source seismic interpretation system, with a plugin architecture that could be extended at run-time with commercial (closed source) and Open source plugins.

OpendTect development is sustained by funding from the industry & governments, maintenance fees collected from commercial users of OpendTect Base (The Open source part) and income from commercial plugins. The functionalities you are about to use have been developed with the funding from:

General Base improvments

Gaz de France improvments:

Jean-Francois Dutzer of Gaz de France proposed (and funded) various improvments to the Open Source part of OpendTect. To name few:
  • Random lines through wells
  • Create New Well track through picking
  • Move element to top/ to bottom
  • Set gridlines option
  • Display horizon at section only
  • Lock/ Unlock elements in the tree
  • Making Colorbars more user friendly


BG General improvments:

Before rolling out OpendTect + plugins to all E&P users within BG, Hamish Macintyre and Charles Jones of the Advanced Geoscience Group created a wish-list with general improvments they'd like to see implemented in OpendTect Base (the open source part). Substantial funds were raised and between end-2007 and mid-2009 dGB implemented many improvments in versions v3.2 and v3.3. To name few:
  • Various visualization improvments
  • New attributes
  • Pre-stack SEGY access
  • SEGY wizard
  • Crossplot / 3D connection
  • Color blending
  • Color bar / Scaling
  • Zero-offset VSP
